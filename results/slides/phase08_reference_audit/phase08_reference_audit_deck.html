TMM Calculation Flow

上半页讲计算流程，下半页说明单点 trace 的数值到底从哪条曲线、哪张材料表取来。

01
nk table
读取对齐后的 Glass、PVK、Ag 光学常数表，并锁定同一波长采样。
02
complex n(λ)+ik(λ)
把每种材料写成复折射率，显式保留吸收项，而不是只看 n。
03
coherent stack
在 Glass/PVK/Air 后侧薄膜栈中计算相位延迟与振幅叠加。
04
front-surface Fresnel
单独求 Air/Glass 前表面 Fresnel 反射，保持与薄膜干涉层分离。
05
incoherent cascade
把 1 mm 玻璃当作非相干隔离层，用强度级联组合前后两部分。
06
R_TMM(λ)
输出可与实验反射率逐点对比的理论曲线。

取值来源：反射率曲线

虚线固定在 599.987 nm，对应单点 trace 中使用的最近数据点。

取值来源：n,k 材料表

同一根虚线在材料表上也落到相同波长，保证代入的 PVK n,k 与反射率 trace 来自同一数据口径。

Risk Source Map

公式链已经过审计，但这不等于实验-理论差异已经被定位到单一原因。

实验链路

低证据 / 低可控性
  • 背景扣除与 ROI 语义
  • glass/Ag 与 Ag mirror 切换一致性
  • 现场标准片与重复性 QC

n,k 适用性

低证据 / 高可控性
  • PVK 文献常数是否代表当前样品
  • Ag 常数与实际镜面状态的偏离
  • 需要 envelope 而不是单点信任

模型边界

高证据 / 低可控性
  • specular-only 假设
  • 厚玻璃非相干近似
  • 显微场景是否可直接等价于理想平面波

显微收光

高证据 / 高可控性
  • NA 与空间平均
  • 散射与非镜面贡献
  • objective / alignment 对实测值的放大效应
当前优先事项：先做 n,k audit + envelope，并补 glass only / standard wafer / field QC，不要把“公式自洽”误判为“物理链路已闭环”。
