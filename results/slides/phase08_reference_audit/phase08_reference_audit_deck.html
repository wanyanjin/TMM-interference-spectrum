Phase 08 反射率校准与 TMM 审计

目标：解释显微反射率如何从原始光谱与参比得到，并说明为何当前实验反射率高于 TMM 理论值。

  • 数据集：0429
  • 当前 audited 结果集：pvk_x01
  • 单点审计波长：599.987374 nm

研究问题：显微反射谱能否得到可 TMM 建模的反射率？

核心矛盾：在 600 nm 附近，实验反射率约为理论值的 2 倍。

  • 实验链：glass/PVK → counts/ms → ratio × R_ref → R_exp
  • 理论链：n,k + layer stack + incoherent glass cascade → R_TMM
  • 本轮不是重新拟合，而是把这两条链完整讲清楚。

实验光路和样品几何

  • 平台：IX73 / IX3-RFAL / 4X objective / NA 0.1
  • 倒置显微镜，从玻璃侧入射并收光
  • 当前建模对象：Air / Glass / PVK / Air

原始光谱和参比对象

  • glass/PVK：样品光谱
  • glass/Ag：同光路参比
  • Ag mirror + bk：直接银镜参比；第 1 帧过曝，后 99 帧可用

实验反射率校准公式

  • 先把 sample / reference 都转成 counts/ms
  • 再做比值，最后乘以参比反射率
  • 反射率单位统一为 0–1

glass/Ag 和 Ag mirror 两条参比链

R_ref = R_TMM_GlassAg

R_ref = R_TMM_AgMirror

  • 两条参比链共享同一份样品 counts
  • 参比结构和理论反射率不同，不能混用

TMM 输入：nk 与层结构

  • nk 来源：aligned_full_stack_nk_phase08_x01.csv
  • wavelength used range：400.213–931.443 nm
  • PVK fixed thickness：700 nm
  • best d：458 nm

n + ik 决定 Fresnel 系数、相位项和吸收强度。

TMM 后侧相干栈：Glass / PVK / Air

  • 后侧薄膜栈是相干问题，不能把界面反射率直接相加
  • 相位项 δ 控制条纹位置和干涉强弱

残差与核心矛盾

  • 600 nm residual by glass/Ag: 0.162179
  • 600 nm residual by Ag mirror: 0.135188

核心矛盾不是公式不自洽，而是实验反射率显著高于理论值。

审计结论：证明了什么，没有证明什么

已证明

  • target wavelength matched to nearest data point
  • Ag first frame excluded
  • Ag frames reduced by mean
  • manual coherent formula vs coh_tmm
  • manual incoherent cascade vs CSV
  • experimental reflectance manual vs CSV

未证明 / 需警惕

  • 当前只审计 output_tag=pvk_x01 对应 nk 来源，不能证明该 nk 全谱适用。
  • 公式自洽不等于显微镜实测必然等于理想 specular reflectance。
